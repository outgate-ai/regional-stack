DETECTION CRITERIA: Authentication Credentials and Secrets

Detect the following types of credentials:
- API keys (sk-, pk-, api_, Bearer tokens)
- Passwords and passphrases
- Access tokens and refresh tokens
- Private keys and certificates
- Database connection strings
- OAuth tokens and secrets

CRITICAL: Extract ONLY the secret value, never include the key name or assignment operator.
- "POSTGRES_PASSWORD=adjkfh98hf" → text should be "adjkfh98hf", NOT "POSTGRES_PASSWORD=adjkfh98hf"
- "VITE_OAUTH_REDIRECT_URI=http://example.com/callback" → text should be "http://example.com/callback"
- "Authorization: Bearer sk-abc123" → text should be "sk-abc123"
- For KEY=VALUE patterns, always extract only the VALUE part after the = sign
